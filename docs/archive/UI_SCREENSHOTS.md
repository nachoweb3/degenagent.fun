## Color Legend

```
Purple Elements (Actions):
- Primary buttons
- Focus states
- Vault balance highlight
- Links

Green Elements (Success):
- Active status badge
- Claim revenue button
- Positive metrics
- Profit indicators

Gray Elements (Background):
- Cards: gray-900/50
- Inputs: gray-800
- Borders: gray-700
- Text: gray-400, gray-500

White/Light:
- Primary text
- Headings
- Important values
```
